# Saidi

Your personal AI assistant / orchestrator.

## Install

```
npm i -g github:saeed9321/saidi-agent
```

Then run:

```
saidi
```

This starts Saidi and opens the dashboard in your browser.

## Updating

Saidi checks for new versions and shows an in-app "Update available" notice.
To update, re-run the install command above in a terminal, then restart Saidi:

```
npm i -g github:saeed9321/saidi-agent
```

This package contains only the built, minified application — no source.
Version: 1.2.0
